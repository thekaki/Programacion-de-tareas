Programacion de tareas

EL programador de tareas es una herramienta de windows que nos ayuda a iniciar o ejecutar tareas automatizadas que realizan acciones a una hora concreta o cuando se produzca un determinado evento.

Actividades que vamos a realizar

Entorno grafico

Vamos a realizar una tarea que diariamente en el inicio de sesion de cualquier usuario nos muestre un mensaje.

Entorno textual

Desde la powershell vamos a realizar una tarea programada que ejecute un script el cual realize una copia de seguridad de manera diaria
